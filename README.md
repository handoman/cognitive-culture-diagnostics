# Cognitive Culture Diagnostics

This repository contains a structured monograph exploring **cognitive culture** as a missing diagnostic dimension in neuropsychiatric assessment.

The work introduces the **Cognitive Culture Framework (CCF)** and applies it through a transparent single-subject methodology to examine why certain high-functioning cognitive profiles—particularly Systemic and Meta-Logical thinkers—are frequently misinterpreted within existing diagnostic systems.

---

## Core Premise

Neuropsychiatric diagnostics currently operate with strong tools for:
- categorical classification (e.g. DSM),
- functional assessment (e.g. executive profiles),

but lack a stable way to describe **how cognition is internally structured**.

This project argues that:

> When diagnostic tools meet a cognitive architecture they were not designed for,  
> it is not the individual who fails — it is the model.

## What This Project Is

- A **theoretical framework** for describing cognitive architecture
- A **methodologically transparent case analysis**
- A **clinical translation** of cognitive culture into assessment practice
- A **research program** proposal for future diagnostics

---

## What This Project Is Not

- Not a new diagnosis
- Not a replacement for DSM
- Not a population-level generalization
- Not a treatment manual
- Not a self-help text

---

## Structure Overview

- **Part I–II**: Theoretical foundations (CCF, SCC, MLCC)
- **Part III**: Applied single-subject analysis (Self-as-Instrument Design)
- **Part IV**: Clinical tools, interviewing, and assessment design
- **Part V**: Epistemology, future diagnostics, and synthesis
- **Appendices**: Terminology, clinical tools, checklists, and models

The full monograph is located in `content/sv/`.  
Clinical reference material is located in `content/sv/appendices/`.

---

## Methodology

The project uses a **Self-as-Instrument Design (SAID)**:
- non-anonymized,
- introspectively rich,
- longitudinal,
- explicitly bounded in scope.

This choice is methodological, not autobiographical.

---

## Language Note

The core monograph is written in **Swedish**, as it is grounded in Swedish clinical practice and diagnostic context.  
Project-level documentation, structure, and methodological framing are provided in **English** to support broader academic accessibility.
