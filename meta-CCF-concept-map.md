# 2. Ontology Root: Cognitive Culture Framework (CCF)

**Definition:**  
CCF är en deskriptiv modell för **intern kognitiv arkitektur** — hur människor tänker, organiserar information och bildar mönster.

**Primära egenskaper:**  
- mönstertäthet  
- konceptuell strukturering  
- systemisk koherens  
- meta‑logisk bearbetning  
- självreflektion  
- icke‑linjär logik  

**Relationer:**  
- CCF → SCC  
- CCF → MLCC  
- CCF → Diagnostisk paradox  
- CCF → Kliniska verktyg  
- CCF → Epistemologi  
- CCF → SAID  

**Answer Unit (AU‑CCF):**  
CCF beskriver *hur* människor tänker (arkitektur), inte *vad* de gör (funktion).

# 3. Cognitive Cultures (Primary Nodes)

## 3.1 Systemic Cognitive Culture (SCC)

**Egenskaper:**  
- hög koherens  
- multi‑lagrad struktur  
- mönsterintegration  
- systemisk logik  

**Relationer:**  
- SCC → CCF  
- SCC → ADHD‑differentiatorer  
- SCC → Kliniska checklistor  
- SCC → Skill trees  

**AU‑SCC:**  
SCC är en systemisk, mönstertät arkitektur som ofta misstolkas som AST eller ADHD.

---

## 3.2 Meta‑Logical Cognitive Culture (MLCC)

**Egenskaper:**  
- abstrakt logik  
- självreferens  
- meta‑nivå resonemang  
- konceptuell precision  

**Relationer:**  
- MLCC → CCF  
- MLCC → ADHD/AST‑differentiatorer  
- MLCC → Kliniska checklistor  
- MLCC → Skill trees  

**AU‑MLCC:**  
MLCC är en meta‑logisk arkitektur som ofta misstolkas i NP‑utredningar.

---

# 4. Diagnostic Paradox (Structural Node)

**Definition:**  
Feltolkning uppstår när diagnostiska verktyg möter en kognitiv arkitektur de inte är designade för.

**Relationer:**  
- Diagnostisk paradox → SCC  
- Diagnostisk paradox → MLCC  
- Diagnostisk paradox → Kliniska verktyg  
- Diagnostisk paradox → Misalignment patterns  

**AU‑DP:**  
Det är inte individen som fallerar — det är modellen.

---

# 5. Methodology Node: Self‑as‑Instrument Design (SAID)

**Definition:**  
En transparent, icke‑anonymiserad, introspektivt rik single‑subject‑metod.

**Egenskaper:**  
- longitudinell  
- metodologiskt avgränsad  
- epistemiskt explicit  

**Relationer:**  
- SAID → Del III  
- SAID → Methodology notes  
- SAID → CCF  

**AU‑SAID:**  
SAID används för att synliggöra arkitektur som inte kan fångas i population‑nivådata.
